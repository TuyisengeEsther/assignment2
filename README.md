# Kigali Explorer - Things to Do & Places to Eat Around Kigali

## Project Title
**Kigali Explorer: Eat, Wander & Feel Kigali**

## Short Project Description
A multi-page website showcasing the best places to eat and activities to do in Kigali, Rwanda. The website features four destination pages (Nyandungu Eco Park, Kimironko Market, Kigali Heights, and Inema Arts Centre) plus a homepage with interactive elements. It serves as a local guide for tourists and newcomers to discover authentic Kigali experiences.

## List of HTML Pages Included
1. **index.html** - Home page with form, table, video, audio, and embedded map
2. **nyandungu.html** - Nyandungu Eco Park destination page
3. **kimironko.html** - Kimironko Market destination page
4. **heights.html** - Kigali Heights rooftop dining page
5. **inema.html** - Inema Arts Centre page
