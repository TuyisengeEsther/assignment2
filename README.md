# Kigali Explorer — Things to Do & Places to Eat Around Kigali

## Project Description
A handcrafted website showcasing four destinations in Kigali: Nyandungu Eco Park, Kimironko Market, Kigali Heights, and Inema Arts Centre. Includes local food tips, tables, forms, video/audio, and embedded maps.

## HTML Pages Included
- index.html (Home with form, table, lists, map, video/audio)
- nyandungu.html (Eco park)
- kimironko.html (Market & food)
- kigali-heights.html (Rooftop dining)
- inema.html (Art centre)

## What This Project Demonstrates
- Semantic HTML5 (header, nav, main, section, footer)
- CSS Flexbox & Grid layouts
- Hover effects on all navigation links
- Internal & external links
- 10+ images across all pages
- Ordered & unordered lists, table, video, audio, iframe map
- Separate CSS files per page with unique class names
- Embedded working form
